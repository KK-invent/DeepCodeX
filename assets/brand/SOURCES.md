# Brand Asset Sources

DeepCodeX uses original vector artwork for the public-facing repository.

## Original DeepCodeX Assets

- `deepcodex-icon.svg`: original vector icon drawn for this repository. It uses a standalone uppercase `X` mark with a cyan-to-blue gradient.
- `deepcodex-hero.svg`: original light hero artwork with a navy wordmark, cyan-to-blue gradient `X`, and soft blue light trails.
- `deepcodex-logo.svg`: original English wordmark composition using navy lettering and the same standalone gradient `X` mark.
- `deepcodex-logo.zh-CN.svg`: Chinese-localized original wordmark composition using navy lettering and the same standalone gradient `X` mark.

No DeepSeek official image asset is tracked in the public-facing asset set. DeepSeek marks and graphics belong to their respective owner.
